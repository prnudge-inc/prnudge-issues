# PRNudge Support

This repository is the public support hub for the **PRNudge** GitHub App.

Use this repo to:
- Report bugs
- Request features
- Ask product questions (optional: via Discussions if enabled)

## Before you open an issue

Please check:
- Existing issues (open and closed)
- Release notes / docs (if applicable)

When filing an issue, include:
- **Installation ID** (from the app installation or logs)
- **Repository full name** (e.g., `org/repo`)
- **Time window** when it happened (include timezone)
- **What you expected** vs **what happened**
- Any relevant logs (redact secrets)

## What not to share

Do **not** post:
- Tokens, secrets, API keys
- Private repository code or sensitive data
- Full payloads if they contain confidential content

If you accidentally shared sensitive info, delete it immediately and open a new issue with redacted details.
